Práctica 5: Galería Dinámica de Imágenes de Perritos

En esta práctica, aprendemos a consumir una API REST para obtener imágenes de perros y mostrar una galería dinámica con textos descriptivos. Además, se incluye un botón para cargar una imagen dinámica adicional que se actualiza mediante un servicio de recarga.

Funcionalidades principales:
  1. Galería inicial de imágenes:
    • Obtención de imágenes desde la API y renderización dinámica con *ngFor.
  2. Imagen dinámica:
    • Carga y refresco de una imagen adicional al presionar el botón o mediante el servicio Reload.
  3. Animaciones y estilos:
    • Animación de aparición con fadeInUp y diseño responsivo con Flexbox.
