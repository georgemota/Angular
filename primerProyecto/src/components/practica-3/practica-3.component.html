Práctica 3: Directivas Estructurales

En esta práctica aprenderás a controlar el flujo de visualización en Angular utilizando directivas estructurales modernas como @if y @for, así como estilos dinámicos con [ngClass]. Se busca mejorar la comprensión de cómo aplicar lógica condicional y repetir estructuras dentro del template de manera eficiente.

Ejercicios e instrucciones:
  1. Mostrar un mensaje condicional con @if.
    • Usar @if para mostrar y ocultar un mensaje basado en una variable booleana.
    • Crear un botón que alterne entre mostrar y ocultar un mensaje como "¡Bienvenido al curso de Angular!".
    • Define una variable isVisible en el componente e inicializada en false.
    • Usa @if para mostrar el mensaje solo cuando isVisible sea true.
  2. Renderizar una lista de elementos con @for.
    • Usar @for para iterar y mostrar una lista de datos.
    • Crear un componente que muestre una lista de frutas: ["Manzana", "Plátano, "Cereza", "Durazno", "Fresa"].
    • Usar @for en HTML para mostrar cada fruta en un <li>.
    • Agrega un botón que permita añadir una fruta nueva al final de la lista.
  3. Usar @if con una condición anidada.
    • Crear un componente que tenga una variable score(number) que represente la puntuación de un estudiante.
    • Si el puntaje es mayor o igual a 90, muestre "Excelente".
    • Si el puntaje está entre 60 y 89, muestre "Aprobado".
    • Si el puntaje es menor a 60, muestre "Reprobado".
    • Usa botones para cambiar dinámicamente el puntaje.
  4. Tablas dinámicas con @for y directivas combinadas..
    • Crear una tabla dinámica para visualizar una listaq de usuarios.
    • Define en el componente una lista de usuarios, donde cada usuario sea un objeto con propiedades: { nombre: string, edad: number, activo: boolean }.
    • Usa @for para iterar y mostrar cada usuario en una fila de una tabla.
    • Agrega una columna extra que muestre Activo o Inactivo según el valor de la propiedad activo, usando @if.
