Práctica 2: Interpolación de Datos

El objetivo de esta práctica es aplicar interpolación para mostrar en pantalla distintos valores, expresiones y llamadas a métodos desde el componente. Se busca fortalecer la comprensión de cómo Angular renderiza datos dinámicos en el template.

Ejercicios e instrucciones:
  1. Interpolación básica.
    • Mostrar un mensaje de bienvenida utilizando interpolación. Mostrar el valor de esta variable en pantalla welcomeMessage: string = 'Bienvenidos a mi aplicación Angular'.
  2. Interpolación con propiedades de clase.
    • Mostrar el nombre y la edad de una persona utilizando interpolación. Ejemplo: TS: person = { name: 'Juan', age: 30 };, HTML: { { person.name } };. Falta mostrar age.
  3. Interpolación con expresiones:
    • Mostrar la resta de num1 = 12 y num2 = 20, utilizando interpolación y expresiones, agregarle texto a la respuesta que vas a mostrar en pantalla.
    • Mostrar la multiplicación de num1 = 8 y num2 = 4, utilizando interpolación y expresiones, agregarle texto a la respuesta que vas a mostrar en pantalla.
  4. Interpolación con métodos:
    • Utilizar un método para calcular y mostrar el resultado de una operación. Ejemplo: TS: radius = 5;, HTML: El área del círculo es { { calculateArea(radius) } }.
    • Ejercicio opcional: Deben usar el método Math.PI y Math.POW.
  5. Interpolación con condiciones:
    • Mostrar un mensaje condicional basado en el valor de una propiedad. Ejemplo: TS: isLoggedIn: boolean = true;, HTML: { { isLoggedIn ? 'Bienvenido de nuevo' : 'Por favor, inicia sesión' } }
  6. Interpolación con pipe:
    • Formatear una fecha utilizando el pipe date. Ejemplo: currentDate: Date = new Date();.
