Práctica Final Angular: Menú Digital

El objetivo de esta práctica es desarrollar una página web utilizando Angular, centrada en un Menú Digital para un restaurante. El proyecto debe integrar funcionalidades interactivas conectadas a la API pública TheMealDB, asegurando una experiencia visual atractiva y profesional.

1. Pantalla Principal (Home)
  • Debe incluir tres botones, cada uno enlazado a una funcionalidad específica (buscar por nombre, por letra y platillo aleatorio).
  • Diseño limpio, organizado y con una estética que simule un menú digital de restaurante.
2. Servicio para consumir la API TheMealDB

Implementar un servicio Angular dedicado para manejar las peticiones HTTP:

  • Petición por nombre: https://themealdb.com/api/json/v1/1/search.php?s=Arrabiata
  • Petición por letra: https://themealdb.com/api/json/v1/1/search.php?f=a
  • Petición aleatoria: https://themealdb.com/api/json/v1/1/random.php
3. Componentes desarrollados
  • SearchByNameComponent: Permite buscar platillos por nombre usando ReactiveForms. Aplica animaciones zoomIn para mostrar resultados.
  • SearchByLetterComponent: Permite buscar por una letra inicial. Incluye spinner y animación en los resultados.
  • RandomMealComponent: Muestra un platillo aleatorio al iniciar o presionar un botón. Usa animaciones fadeOut y fadeIn.
4. Estructura y servicios
  • Se centraliza la lógica en un servicio llamado MealService.
  • Los ingredientes se procesan mediante una función reutilizable por componente.
5. Sistema de carga global
  • Se utiliza un spinner de carga en todos los componentes mientras se reciben datos e imágenes.
  • El contenido se oculta hasta que todo esté listo, usando animaciones suaves de entrada.
6. Diseño visual
  • Colores personalizados en tonos marrón usando la clase .bg-food.
  • Tipografía de Google Fonts (Rubik) y diseño responsive con Bootstrap.
  • Animaciones de animate.css como zoomIn, fadeOut y zoomOut.
  • Fondo texturizado para simular papel o menú impreso, integrado con background-image.
7. Extras implementados
  • Pantalla de bienvenida (splash screen): imagen con animación zoomIn y salida zoomOut.
  • Manejo de errores: muestra alerta Bootstrap con color personalizado si no se encuentran resultados.
