Final Angular Project: Digital Menu

The goal of this project is to develop a website using Angular, focused on a Digital Menu for a restaurant. The project should integrate interactive features connected to the public API TheMealDB, ensuring a visually appealing and professional user experience.

1. Home Screen
  • It must include three buttons, each linked to a specific feature (search by name, by letter, and random dish).
  • Clean, organized design with an aesthetic that simulates a restaurant's digital menu.
2. TheMealDB API Service

Implement a dedicated Angular service to handle HTTP requests:

  • Search by name: https://themealdb.com/api/json/v1/1/search.php?s=Arrabiata
  • Search by letter: https://themealdb.com/api/json/v1/1/search.php?f=a
  • Random dish: https://themealdb.com/api/json/v1/1/random.php
3. Developed Components
  • SearchByNameComponent: Allows searching for meals by name using ReactiveForms. Applies zoomIn animations to display results.
  • SearchByLetterComponent: Allows search by initial letter. Includes a loading spinner and result animation.
  • RandomMealComponent: Displays a random dish on startup or when a button is pressed. Uses fadeOut and fadeIn animations.
4. Structure and Services
  • The logic is centralized in a service called MealService.
  • Ingredients are processed using a reusable function within each component.
5. Global Loading System
  • A loading spinner is shown in all components while waiting for data and images.
  • Content remains hidden until everything is ready, then appears with smooth entrance animations.
6. Visual Design
  • Custom brown-themed colors using the .bg-food class.
  • Typography from Google Fonts (Rubik) and responsive design with Bootstrap.
  • Animations from animate.css such as zoomIn, fadeOut, and zoomOut.
  • Textured background to simulate paper or printed menu, using background-image.
7. Extra Features
  • Welcome screen (splash screen): image with zoomIn animation and exit zoomOut.
  • Error handling: shows a Bootstrap alert with custom color if no results are found.
